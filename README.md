# 👻 Phantomix

### The open-source AI browser agent. Free alternative to OpenAI Operator.

Phantomix lets Claude control your browser and do things for you automatically. Browse, scrape, fill forms, book appointments, shop online — all via Telegram or a web dashboard.

## ✨ Why Phantomix?

- 🆓 100% free and open-source — self-host in minutes
- 🤖 Claude-powered — best AI model for reasoning and tasks
- 📱 Control via Telegram — send tasks from your phone anywhere
- 🌐 Web dashboard — visual interface to monitor everything
- 🧠 Persistent memory — remembers your preferences
- 🔑 Bring your own API key — you control your costs
- 🐳 One command launch — Docker makes it dead simple

## 🚀 Quick Start

1. Clone the repo: `git clone https://github.com/dimitrisdimitrov5-blip/Phantomix`
2. Copy env file: `cp .env.example .env`
3. Add your API keys to `.env`
4. Launch: `docker-compose up -d`

## 💬 Example Tasks

- "Search for the latest AI news and summarize it"
- "Go to amazon.com and find the best laptop under $1000"
